
Chào mừng bạn đến với tài liệu dự án Directus CMS Starter -- SvelteKit.

## Ngữ cảnh dự án

- **Loại site:** CMS template / starter kit cho trang web sử dụng Directus làm headless CMS
- **Người dùng cuối:** Developer có thể tùy chỉnh để làm landing page, blog, trang marketing công ty, trang bán sản phẩm
- **Ngôn ngữ nội dung:** Tiếng Việt / Tiếng Anh (tùy biến)
- **Ghi chú đặc biệt:** Đây là Directus starter template (origin: directus-labs/starters), fork về và tùy chỉnh

## Cấu trúc thư mục tài liệu

```
docs/
├── designs/       # <tên>.design.md -- tài liệu thiết kế, brainstorm
├── specs/         # <tên>.spec.md   -- đặc tả chi tiết tính năng
├── plans/         # <tên>.plan.md   -- kế hoạch thực hiện
├── overviews/     # <tên>.overview.md   -- tổng quan tính năng
├── ROADMAP.md     -- theo dõi tiến độ tất cả tính năng
├── CONVENTIONS.md -- quy ước code
├── STACK.md       -- công nghệ sử dụng
└── Welcome.md     -- giới thiệu tài liệu
```
